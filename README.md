# QuizMate – Online Quiz Platform

QuizMate is a full-stack web application that allows administrators to create quizzes and users to attempt them online.

## Features
- User Registration & Login
- Admin Panel to Create Quizzes
- Multiple Choice Questions
- Quiz Result Tracking
- MySQL Database Integration
- Responsive UI

## Technologies Used
- Node.js
- Express.js
- MySQL
- EJS
- Tailwind CSS
- HTML, CSS, JavaScript

## Installation

Clone the repository

git clone https://github.com/Sailaja505/QuizMate.git

Install dependencies

npm install

Start the server

node app.js

Open in browser

http://localhost:3000
